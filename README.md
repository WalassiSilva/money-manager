# Gerenciador Financeiro

Esse Projeto tem como intuito, ser um gerênciador financeiro pessoal.
Nele será possível:
- Criar transação;
- Alterar transação;
- Deletar transação;
- Visualizar transações de forma dinâmica
  - Por mês (x)
  - Por Categorias (x)
  - Ver categorias com gráfico (x)
  - Pesquisar por nome
